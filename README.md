# Labs

## Setup

Pull down repo

```bash
git clone https://github.com/jasonmerino/labs.git
```

Enter directory

```bash
cd labs
```

NPM Install

```bash
npm i
```

Install Bower

```bash
npm i -g bower
```

Install Bower components

```bash
bower install
```

## Project Layout

For client-side libs use Bower.
For server-side, testing, and build tools use NPM.
